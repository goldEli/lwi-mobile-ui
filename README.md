# 跨框架组件库

兼容 tarojs 和 reactjs 跨框架组件库

> 由于 gitee pages 在维护中，官方说需要维护几个月，所以文档暂时托管到 github pages。查看可能需要翻墙

[在线文档](https://goldeli.github.io/lwi-mobile-ui)

## Development

```sh
yarn dev:rollup
```

```sh
yarn dev:lib
```

h5，only for storybook

```sh
yarn dev:rollup-h5
```

## Production

```sh
yarn build
```
